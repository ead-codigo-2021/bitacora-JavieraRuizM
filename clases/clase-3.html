JavaScript

JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web

Preprocesadores CSS

Un preprocesador de CSS se puede definir como una herramienta que nos permite escribir pseudocódigo CSS que luego será compilado de convertir en CSS tal y como lo conocemos de forma habitual.

Anidación

Es posible anidar los selectores de los elementos descendentes, lo que permite visualizar mejor la jerarquía de los elementos. Debe utilizarse cuidadosamente, ya que la especificidad de los elementos puede aumentar sin darnos cuenta.

Variables

Tal como las variables CSS, cada preprocesador cuenta con la alternativa de crear variables reutilizables.

Operaciones

Otra cualidad que consideran casi todos los preprocesadores es poder realizar operaciones matemáticas dentro del valor de una propiedad.

Funciones

Sass ofrece una larga lista de funciones que podemos utilizar para gestionar colores, hacer cálculos o programar funciones más complejas

Import

Una de las principales ventajas de un preprocesador es la capacidad de importar otros archivos scss (o css), lo que permite aislar estilos en hojas separadas.

Mixins

Forma de crear funciones propias, para agrupar estilos comunes que puedan generarse a partir de variables. Podemos definir los parámetros y comportamientos internos a partir de ellos.
